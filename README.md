# RelayTerminalLocalization

Synopsis

IMPROVING THE PERFORMANCE OF MOBILE DATA GATHERING BY THE LOCALIZATION OF RELAY TERMINAL IN THREE LAYER APPROACH


Programs needed to Run the project

network simulator

cd script
ns main.tcl

gnuplot graph1.plot
